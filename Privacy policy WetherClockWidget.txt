############ Privacy Policy for WeatherClockWidget ############ 

Personal information:
This application does not collect any personal information about users of my application.

Geo-Location:
This application, when you turn on the "GPS Location detection" setting determines the location of your mobile device. This information serves only to download weather forecast data. Only information about the current location is stored on the device. When you get new coordinates, it is overwritten.

Access to the Internet:
This application, if there is an Internet connection, downloads the weather forecast from the Internet resource http://api.openweathermap.org

Links in applications
This application does not contain any external links.

Advertising
This application does not contain any third-party advertising.

Личная информация:

Гео-Location:
